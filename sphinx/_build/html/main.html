main module¶

main.py¶

This module contains the main functions to extract comments and relative text from word files and convert them into EAI Platform annotations

main.create_annotation_and_text_file(doc_file_path: Path, comments_dicts: list, folders: dict, highlight_mode: bool = True)[source]¶

For each comment in comment_dicts create a corresponding annotation and test file in their relative folders

Parameters:

  • doc_file_path – path of a single word document (.docx)

  • comments_dicts – list of comment dictionaries generated by main.return_comments_dicts()

  • folders – folder structure generated by platform_utils_eai.functions.create_folder_structure()

  • highlight_mode – if true, for every comment in the current file, create an ann file with the annotation for the text highlighted by the comment. This works better for autoML, else create a single ann file for the current file with all the annotations for all the comments

main.main()[source]¶
main.return_comments_dicts(doc_file_path: Path) → Union[list, bool][source]¶

In a word file, for every comment in it creates a dict with the text of the comment, the text on which the comment was placed, the start character of the comment and the end character of the comment. Then returns either a list of dictionaries (one dictionary per comment in the file), or False if no comment was found in the docx.

Parameters:

doc_file_path – path of a single word document (.docx)

Returns:

list of dictionaries (one for each comment) or False
